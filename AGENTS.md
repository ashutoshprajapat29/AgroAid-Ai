# Project Guidelines & Architecture
- **Scalability:** The app must be scalable to handle 100,000 (1 lakh) users at any point in time.
- **Cost Efficiency:** Strictly use free tiers for all services (Firebase, Supabase, APIs, etc.). We will upgrade later only if absolutely necessary.
- **Localization (i18n):** The app must support and be fully compatible with both Hindi and English languages.
- **Mobile-First Design:** This web app will eventually be converted to a mobile app (iOS/Android). Design everything with a mobile-first approach (touch-friendly buttons, responsive layouts, avoid hover-only menus).
- **SEO Optimization:** Implement proper SEO best practices (meta tags, semantic HTML) to ensure the app ranks high in Google Search results.
